How to: Adjust Settings Related to Visible Group Headers and Overflow Panel in NavigationPaneView

This example demonstrates how to adjust Visible Group Headers (https://documentation.devexpress.com/#WPF/CustomDocument6696) and Overflow Panel (https://documentation.devexpress.com/#WPF/CustomDocument6686) in NavBarControl (https://documentation.devexpress.com/#WPF/clsDevExpressXpfNavBarNavBarControltopic) when NavigationPaneView is used. This example demonstrates how to adjust Visible Group Headers (https://documentation.devexpress.com/#WPF/CustomDocument6696) and Overflow Panel (https://documentation.devexpress.com/#WPF/CustomDocument6686) in NavBarControl when NavigationPaneView is used. NavigationPaneView provides the following properties for this purpose: -IsSplitterVisible (https://documentation.devexpress.com/#WPF/DevExpressXpfNavBarNavigationPaneView_IsSplitterVisibletopic). Gets or sets whether the splitter above Visible Group Headers (https://documentation.devexpress.com/#WPF/CustomDocument6696) is visible -IsOverflowPanelVisible (https://documentation.devexpress.com/#WPF/DevExpressXpfNavBarNavigationPaneView_IsOverflowPanelVisibletopic). Gets or sets whether the Overflow Panel is visible -MaxVisibleGroupCount (https://documentation.devexpress.com/#WPF/DevExpressXpfNavBarNavigationPaneView_MaxVisibleGroupCounttopic). Specifies the maximum number of visible group headers in the Group Button Panel (https://documentation.devexpress.com/WPF/CustomDocument6696.aspx) -GroupButtonTemplate (https://documentation.devexpress.com/#WPF/DevExpressXpfNavBarNavigationPaneView_GroupButtonTemplatetopic). Specifies the template applied to items in the Group Button Panel (https://documentation.devexpress.com/WPF/CustomDocument6696.aspx) -OverflowGroupTemplate (https://documentation.devexpress.com/#WPF/DevExpressXpfNavBarNavigationPaneView_OverflowGroupTemplatetopic). Specifies the template applied to items in the Group Button Panel (https://documentation.devexpress.com/WPF/CustomDocument6696.aspx)
